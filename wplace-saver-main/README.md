# Wplace saver

A simple Tampermonkey script to capture [Wplace.live](https://wplace.live) images within a specific range

> [!NOTE]
> You need to use it with [Blue Marble](https://github.com/SwingTheVine/Wplace-BlueMarble) or other script that can get the coordinates of a pixel.

## Run locally
```
pnpm i
pnpm run dev
```

### Generate user.js
```
pnpm run build
```
